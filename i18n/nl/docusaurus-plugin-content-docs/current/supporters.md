## Waar worden onze Patreon-fondsen voor gebruikt?

Onze Patreon zal worden gebruikt voor financiering van,
- Server Onderhoud
- Toekomstige hardware upgrades
- Ontwikkelings kosten
- Discord Bots
- Website kosten
